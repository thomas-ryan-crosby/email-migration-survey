Outlook on the Web
Heads up

Outlook on the Web isn't the right tool here

Short version: if you want to read your Crosby Workspace email in a browser, use gmail.com — not outlook.com.

Why?

Outlook on the web (the website at outlook.com or outlook.office.com) is Microsoft's webmail. It's built to show email that lives on a Microsoft mail server.

Your Crosby email now lives on a Google mail server (Google Workspace). The Google webmail at gmail.com is the matching tool — same company runs both ends.

Yes — technically you can add a Gmail account to outlook.com via Microsoft's "Connected accounts" feature, but it's clunky, often breaks with 2FA, and Microsoft has been deprecating that feature. We don't recommend it.

What to use instead

  • Want to check email in a browser on a computer? Go to gmail.com. See the Gmail web guide for sign-in steps.
  • Want a desktop app that looks and feels like Outlook? Use the new Outlook for Windows or classic Outlook for Windows — both work great with your Crosby Workspace account.
  • On a Mac? Use Apple Mail on Mac or the desktop Outlook app for Mac.
